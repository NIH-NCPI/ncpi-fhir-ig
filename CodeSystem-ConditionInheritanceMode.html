CodeSystem: Condition Inheritance Pattern

Official URL: https://ncpi-fhir.github.io/ncpi-fhir-ig/CodeSystem/ConditionInheritanceMode Version: 0.1.0
Active as of 2022-05-27 Realm: Computable Name: ConditionInheritanceMode

A set of terms that describe the transmission pattern of a condition in a pedigree.

This Code system is referenced in the content logical definition of the following value sets:
